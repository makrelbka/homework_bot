# Homework Bot

## 📋 Описание  
Бот для записи домашних заданий и хранения метриалов(ссылок) на необходимые данные.  
Поддерживает роли **пользователей** и **администратора** с различными функциями.  

---

## ⚙️ Возможности бота  
### Для пользователей:
- **`/start`** — вывод меню взаимодействия с ботом.  
- **Кнопка "ДЗ"** — просмотр текущего списка домашних заданий.  
- **Кнопка "Ссылки"** — получение сохранённых ссылок.  
- **Кнопка "Предложить изменения"** — отправка предложений по домашним заданиям.  

### Для администратора:
- Все возможности пользователя, а также:  
  - **`/edit`** — выбор предмета для редактирования данных.  
  - **`/clear_data`** — очистка всех данных.  
  - **Кнопка "Очистка предложений"** — удаление списка пользовательских предложений.  

---

## 📂 Структура данных  
- **ДЗ**: хранится как словарь `{предмет: задание}`.  
- **Ссылки**: список полезных ссылок.  
- **Предложения**: список изменений, отправленных пользователями.  

---

## 🛑 Примечания  
- **Администратор** определяется через ID, прописанный в `data.py`.  
- **Пользовательские предложения** требуют проверки администратора перед добавлением.  

---

## 📖 Пример взаимодействия  
1. Пользователь отправляет `/start`, видит меню с кнопками:  
   - "ДЗ" — для просмотра списка заданий.  
   - "Ссылки" — для отображения полезных ссылок.  
   - "Предложить изменения" — для отправки предложения.  
2. Администратор получает возможность:  
   - Просматривать и редактировать данные.  
   - Управлять предложениями через кнопки.

## Развитие проекта
1. Сделать добавление материалов(ссылок) через интерфейс бота.  
2. Добавить напоминания о грядущем дедлайне.  
3. Сделать парсер дат для пункта номер два, и, как следствие, проверку пользовательских сообщений на её наличие.
